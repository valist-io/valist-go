# Valist Go SDK.

## Documentation

Documentation for how to get started with Valist can be found at [https://docs.valist.io](https://docs.valist.io).

## Building

#### System Requirements:

* go >= 1.16

```bash
git clone https://github.com/valist-io/valist-go

cd valist-go

go build .
```
